## ✨ Features

### 💬 Open Career Q&A Chatbot
- Ask anything about careers, skills, salaries, and education
- Powered by Claude AI for intelligent, personalized responses
- Three modes: Career Guidance, Resume Tips, Interview Coach

### 🧠 Career Assessment
- 10-question intelligent career profiling quiz
- Covers Education, Interests, Skills, Personality, and Goals
- AI-generated personalized career match with detailed analysis

### 📄 Resume Analyzer
- Paste or upload resume for instant AI analysis
- ATS Score and Industry Readiness Score
- Missing skills detection, grammar suggestions, improvement tips
- Suitable job title recommendations

### 🗺️ Learning Roadmap
- Step-by-step personalized learning plans
- 6 career paths with phases, topics, tools, and timelines
- Covers AI/ML, Frontend, Data Science, Full Stack, Product, Cybersecurity

### ⚡ Skill Gap Analysis
- Add your current skills and select a target role
- AI compares your profile against industry requirements
- Visual progress bars with priority-based recommendations

### 🎤 Interview Preparation
- Mock interviews for 6 different roles
- Three types: Technical, HR Round, Behavioral
- AI feedback after each answer + final performance summary

## 🚀 Getting Started

### View Online
```
https://stackpilot-00.github.io/career-guidance-chatbot
```

### Run Locally
1. Download `index.html`
2. Open in any browser — no installation needed!

### Enable AI Features
1. Get a free API key at [console.anthropic.com](https://console.anthropic.com)
2. Click **🔑 API Key** in the sidebar
3. Paste your key → all AI features activate!
